3.1 Tareas

En CRM ACAP, las tareas representan acciones que debe realizar un usuario. Realizar un seguimiento de las tareas ayuda a organizar el proceso de trabajo y garantizar que se realice todo el trabajo importante.

Dentro de la aplicación ACAP CRM, se pueden crear y asignar tareas (a otras personas o a uno/a mismo/a), organizarlas por prioridad, setear la fecha de vencimiento, etc. Además, para poder realizar un mejor seguimiento, las tareas pueden tener diferentes estados: abierta (open), en progreso (in progress), o cerrada (closed):

3.2 Llamadas

Las actividades de llamada nos permiten registrar todas las llamadas que realizamos a las organizaciones y contactos relacionados. Si el contacto ya tiene un teléfono registrado, entonces directamente se selecciona su teléfono, aunque también es posible registrar llamadas a números que no estén registrados aún. De todos modos, esta opción no es recomendable ya que es importante contar siempre con todos los datos de los contactos de las organizaciones.
